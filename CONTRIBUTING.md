# Contributing to Fitness Chat App

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/fitness-chat-app.git`
3. Create a feature branch: `git checkout -b feature/my-feature`

## Development Setup

### Backend
```bash
cd backend
npm install
cp .env.example .env
npm run dev
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

## Code Style

- Use TypeScript for type safety
- Follow existing code patterns
- Format code with Prettier (if configured)
- Write meaningful commit messages

## Making Changes

1. Make your changes in a feature branch
2. Test locally
3. Commit with clear messages: `git commit -m "Add feature: description"`
4. Push to your fork
5. Create a Pull Request

## Pull Request Process

1. Provide a clear description of changes
2. Link any related issues
3. Ensure code builds without errors
4. Update README if needed
5. Wait for review

## Reporting Issues

Include:
- Description of the issue
- Steps to reproduce
- Expected vs actual behavior
- Environment info (OS, Node version, etc.)

## Questions?

Open a GitHub discussion or issue. We're here to help!
